Builder Release 8.4.0

** Release Highlights **
- HIGHLIGHTS: New option to disable browser context menu.
- HIGHLIGHTS: Build Mode selection can now be saved.

** Breaking Changes **
- BREAKING: This version will not support saves from previous versions.
- BREAKING: This version only support 6.6.0 or newer resource.

** Release notes **
- ADDED: Flush log will also clear current logged content.
- ADDED: New option to hide Navigation Bar in View Menu.
- IMPROVEMENT: Updated CefSharp to 133.4.21
- IMPROVEMENT: Updated Chromium to 133.0.6943.127
- IMPROVEMENT: Minimal User Interface improvements.
- FIXED: Status will show as not ready when resource not available.
